# 非功能需求

## 性能指标

### 延迟 SLA

| 操作 | P50 | P95 | P99 | 说明 |
|------|-----|-----|-----|------|
| 消息发送 → 对方收到 | < 200ms | < 500ms | < 1s | 端到端消息延迟 |
| VE 收到消息 → 意图识别完成 | < 1s | < 2s | < 3s | 意图分析延迟 |
| 意图识别完成 → 首字回复 | < 3s | < 8s | < 15s | 包含 LLM 推理时间 |
| 工具调用 → 结果返回（本地 WEN） | < 500ms | < 2s | < 5s | 不含 LLM 推理，仅工具执行 |
| VE 冷启动（Suspended → Idle） | < 5s | < 10s | < 15s | 从持久化恢复 |
| 协作应用页面加载 | < 1s | < 2s | < 3s | Flutter 首屏渲染 |

## 可用性

### SLA 目标

| 组件 | 可用性 | 最大不可用时间/月 | 说明 |
|------|--------|-----------------|------|
| 协作应用服务端 | 99.9% | 43 min | 用户消息收发入口 |
| Agent 服务器 | 99.5% | 3.6 h | VE 管理，降级不影响 IM |
| VE Runner | 99.5% | 3.6 h | 单 Runner，VE 可跨 Runner 恢复 |
| 工作环境节点（云端） | 99.5% | 3.6 h | 平台托管节点 |
| LLM API（依赖） | 99% | 7.2 h | 依赖外部服务，有降级策略 |

### 降级模式

| 场景 | 降级行为 | 用户感知 |
|------|---------|---------|
| Agent 服务器不可用 | VE 显示离线，协作应用 IM 正常 | "销售分析师已离线" |
| VE Runner 不可用 | VE 在其他 Runner 恢复（冷启动延迟） | 短暂不可用后自动恢复 |
| LLM API 不可用 | VE 回复"服务暂时不可用，稍后重试" | 明确告知，避免静默失败 |
| 工作环境节点离线 | VE 平台工具可用，远程工具报错 | "工作环境已断开，请检查" |
| PostgreSQL 主库不可用 | 读切换至副本，写操作等待 failover | 消息发送可能短暂失败 |

## 安全合规

### 加密标准

| 范围 | 标准 | 说明 |
|------|------|------|
| 传输加密 | TLS 1.3 | 所有外部通信 |
| 内部通信 | mTLS 1.3 | 服务间通信 |
| 密码存储 | bcrypt (cost ≥ 12) | 用户密码哈希 |
| API Key 存储 | SHA-256 哈希 | 仅存储哈希值 |
| 文件加密 | AES-256-GCM（可选） | 云端工作环境中的敏感文件 |

## 数据保留

| 数据类型 | 默认保留期 | 企业版可配置 | 说明 |
|---------|-----------|------------|------|
| 消息 | 永久 | 可配置 | 用户可手动删除 |
| 已删除消息 | 30 天（软删除） | 可配置 | 30 天后物理删除 |
| 工作上下文（归档） | 90 天 | 可配置（最长 365 天） | 过期后归档摘要保留 |
| 审计日志 | 90 天（热）+ 365 天（冷） | 可配置 | 合规要求可能更长 |
| 文件附件（未引用） | 30 天 | 可配置 | 无消息引用的孤立文件 |
| 不活跃租户 | 180 天未登录 → 暂停，365 天 → 删除 | 可配置 | 提前邮件通知 |
