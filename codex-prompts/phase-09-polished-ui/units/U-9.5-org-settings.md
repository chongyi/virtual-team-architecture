# U-9.5 组织管理与个人设置

## 目标 (Goal)

重塑组织管理页面（组织列表/详情/成员管理/VE 管理）和个人设置页面（个人资料/通知偏好/语言切换/主题切换），使其达到企业应用的管理体验标准。

## 上下文 (Context)

- 前置：U-9.4（通讯录完成）
- 当前组织管理 UI 功能完整但视觉简陋

## 工作范围

### 1. 组织列表 — `org_list_screen.dart`

- 顶部：搜索框 + 创建组织按钮（FAB 或右上角图标）
- 组织卡片（`OrgCard`）：组织图标（首字母）+ 名称 + 角色标签（Owner/Admin/Member，不同颜色）+ 成员数
- 点击进入组织详情

### 2. 组织详情 — `org_detail_screen.dart`

- 顶部区域：组织名称 + 描述 + 成员总数
- 管理员可见：邀请成员按钮 + 管理按钮
- 成员列表：头像 + 名称 + 角色标签（可点击变更角色弹窗）
- 虚拟员工列表（如有绑定 VE）：机器人图标 + VE 名称 + 状态标签（在线/离线）

### 3. VE 管理 — `ve_list_screen.dart`

- VE 卡片：大机器人头像 + VE 名称 + 配置包名称 + 状态（Active/Paused/Offline）
- 管理员操作：创建 VE → 进入 VE 创建表单（名称/配置包选择/环境选择，使用下拉选择器而非 mock 4 个选项）
- 点击 VE 进入详情（当前可跳转到私信）

### 4. 个人设置 — `profile_screen.dart`

页面分组（`ListView` with section headers）：

**个人信息**：
- 头像（大，可点击更换） + 用户名 + 部门
- 点击进入个人资料编辑页

**偏好设置**：
- 语言切换（中文/English，`ListTile` + 右侧当前语言标签）
- 主题切换（浅色/深色/跟随系统，`ListTile` + `SegmentedButton` 或下拉）
- 推送通知开关（`SwitchListTile`，每个类型独立开关：消息通知/@提及通知/审批通知）

**安全**：
- 修改密码（跳转修改密码页）

**关于**：
- 应用版本号 + 隐私政策 + 用户协议 + 开源许可（WebView 或模态框）

**退出登录**：红色文字按钮，确认弹窗。

### 5. 租户切换 — `tenant_switch_screen.dart`

- 当前租户高亮 + 勾号
- 其他租户列表（名称 + 成员数）
- 点击切换 → loading → 成功跳转首页
- 错误时 SnackBar 提示
