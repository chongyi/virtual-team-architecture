# U-9.6 协作工具 UI 全面重塑

## 目标 (Goal)

将 5 类协作工具的 Flutter 界面从功能可用级别提升到美观专业的水平：文档编辑器、多维表格、看板、审批、日程日历。每类工具都要有清晰的视觉层级、一致的交互模式、完善的操作反馈。

## 上下文 (Context)

- 前置：U-9.5（组织设置完成）
- 当前 5 类工具 UI 功能完整（CRUD 都通），但视觉粗糙

## 工作范围

### 1. 文档工具 — `document/`

**文档列表页**：
- 卡片网格布局（移动端 2 列，平板 3 列，桌面 4 列）
- 每张卡片：文档图标/缩略图 + 标题（`title2`）+ 更新时间（`caption2`）+ 作者头像
- 顶部搜索栏 + "新建文档" FAB（右下角）
- 下拉刷新

**文档编辑页**：
- 顶部栏：返回按钮 + 文档标题（可点击编辑）+ 更多操作菜单（归档/版本历史）
- 内容区：垂直滚动的 Block 列表，每个 Block 左对齐
- Block 类型视觉区分：
  - `paragraph`：标准正文（`body1`），行高 1.6
  - `heading`：大号粗体（`headline3`），上下 16px 间距
  - `bullet_list`：左侧圆点 + 缩进 24px
  - `code_block`：等宽字体，灰色背景（`bgSecondary`），圆角 8px，内边距 16px，顶部语言标签
  - `image`：全宽图片，圆角 8px
- 点击 Block 弹出编辑工具栏（粗体/斜体/代码/标题/列表）
- 新建 Block：段落底部 `+` 按钮（hover 时显示，类似 Notion）

**版本历史面板**：
- 右侧滑入面板
- 版本列表：版本号 + 时间 + 编辑者
- 点击版本预览该版本的文档内容（只读）

### 2. 多维表格 — `bitable/`

**表格列表页**：同文档列表页风格，卡片布局。

**表格视图页**：
- 使用 `SingleChildScrollView`（水平+垂直滚动）
- 列头行：列名 + 列类型图标 + 列宽调整手柄 + 列菜单（编辑列/删除列/排序/隐藏）
- 数据行：斑马纹（奇偶行交替 `bgPrimary`/`bgSecondary`，4px 间隔）
- 单元格：点击进入编辑模式（按列类型匹配控件）：
  - `text`：`TextField` 内联
  - `number`：数字键盘
  - `select`：下拉选择器
  - `date`：日期选择器（`showDatePicker`）
  - `checkbox`：勾选框
- 底部：+ 添加行按钮

**视图配置栏**（表格顶部）：
- 过滤器按钮 → 弹出过滤条件编辑面板
- 排序按钮 → 弹出排序条件编辑面板
- 分组按钮 → 选择分组列
- 当前激活的过滤/排序条件以 `Chip` 标签显示，可单独删除

### 3. 看板 — `board/`

**看板详情页**：
- 水平滚动的列（`SingleChildScrollView` horizontal）
- 每列：列标题栏（`title2`，含卡片数量） + 卡片列表（垂直滚动）
- 列间：8px 间距，列宽 280px

**卡片样式**：
- 白色背景 + `shadowSm` + 圆角 8px
- 卡片内容：标题（`body2`）+ 标签行（彩色小圆角标签）+ 负责人头像（右上角）+ 截止日期图标（如有）
- 标签颜色语义：功能（蓝色）、Bug（红色）、改进（绿色）、设计（紫色）、文档（黄色）
- 点击卡片：弹出详情面板（编辑标题/描述/标签/负责人/移动列/删除）

**泳道模式**：
- 分组标题行（按负责人分组）
- 每个泳道：负责人头像 + 名称 + 该泳道内的列+卡片

**创建卡片**：底部 `+ 添加卡片` 按钮，点击弹出快速创建表单：标题（必填）+ 标签选择（可选）+ Enter 确认。

### 4. 审批 — `approval/`

**审批列表页**：
- 顶部分段：待我审批 / 我发起的 / 全部
- 审批卡片：审批标题 + 发起人头像 + 状态标签（审批中-蓝色/已通过-绿色/已驳回-红色/已转交-橙色）+ 发起时间
- 按时间降序排列

**审批详情页**：
- 顶部：审批标题 + 状态大标签 + 发起人信息
- 中部：审批节点时间线（垂直时间线组件）：
  - 每个节点：节点图标（审批中=灰色圆/已通过=绿色勾/已驳回=红色叉）+ 节点名称 + 审批人 + 时间
  - 节点间竖线连接
- 底部操作栏（仅审批人可见）：通过按钮（绿色）+ 驳回按钮（红色，可选填写原因）+ 转交按钮（选择转交人）

**创建审批模板**（管理员）：
- 表单：模板名称 + 节点列表（可添加/删除/排序节点）
- 每个节点：节点名称 + 审批人选择（搜索用户）

### 5. 日程 — `schedule/`

**日历视图**：
- 顶部月份导航：`< 2026年5月 >` + 今日按钮
- 月视图网格：7 列（日一二三四五六），每格含日期数字 + 当天事件小圆点（最多显示 3 个，+N 更多）
- 周视图（可选切换）：水平 7 列，每小时一行
- 点击日期：下方展开该日事件列表

**事件卡片**（在列表中）：
- 左侧颜色条（4px，事件颜色：蓝/红/绿/紫/黄）
- 标题 + 时间范围 + 地点（可选）+ 重复规则图标（如有）
- 点击进入事件详情

**事件详情弹窗**：
- 标题 + 时间范围 + 重复规则（文字描述：每天/每周/每月）
- 关联频道（事件提醒发送到哪个频道）
- 编辑/删除操作

**提醒列表**：
- 提醒卡片：提醒文字 + 触发时间 + 状态（待触发/已触发）
- 手动触发按钮（测试用）

**创建事件**：
- FAB → 弹出表单：标题 + 开始/结束时间 + 重复规则下拉 + 关联频道选择 + 颜色选择

## 约束 (Constraints)

- 全部使用 U-9.1 设计 Token
- 工具列表页保持一致的卡片风格（尺寸/间距/圆角/阴影统一）
- 每个工具页面含空状态组件（U-9.1 的 `AppEmptyState`）
- 每个工具页面含加载和错误状态（U-9.1 的 `AppLoading`/`AppErrorWidget`）
