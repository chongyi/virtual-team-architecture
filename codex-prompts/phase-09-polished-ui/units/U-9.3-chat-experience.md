# U-9.3 聊天体验重塑

## 目标 (Goal)

全面重塑 IM 聊天体验，使其达到企业微信/Slack 级别的交互品质：时间线式消息气泡、富文本渲染、输入框体验、消息操作（复制/回复/反应/转发）、线程视图、附件预览，全部使用 U-9.1 设计 Token。

## 上下文 (Context)

- 前置：U-9.2（导航壳完成）
- 当前消息气泡渲染简陋（纯文本拼接），无时间分组，无滑动操作

## 工作范围

### 1. 消息时间线 — `message_list.dart`

**时间分组**：相邻 5 分钟内的消息归为同一时间组，每组顶部显示时间分隔符（居中，`caption2` 字号，`textTertiary` 颜色）。

**消息布局规则**：
- 自己的消息：右对齐，气泡背景 `colorPrimaryLight`（#E8F0FE），文字 `textPrimary`
- 他人的消息：左对齐，气泡背景 `bgSecondary`（#F8F9FA），文字 `textPrimary`
- VE 的消息：左对齐，气泡背景含细微紫色渐变或特殊边框（1px `colorPrimary` 30% 透明度），头像右下角小机器人图标

**消息气泡样式**：
- 圆角 8px（`radiusMd`）
- 内边距：上下 10px，左右 14px
- 单条消息最大宽度 70%（移动端）/ 60%（桌面端）
- 气泡下方 2px 处：时间戳（`caption2`，`textTertiary`），右对齐

**消息分组气泡（同一个人连续多条）**：
- 第 1 条显示头像 + 名称 + 气泡
- 第 2+ 条仅显示气泡，不显示头像和名称
- 同组气泡间距 2px，组间间距 16px

### 2. 消息气泡内容渲染 — `message_bubble.dart`

每种内容块（ContentBlock）有对应渲染器：

**Text 块**：
- 使用 `flutter_markdown` 的 `MarkdownBody`（已在 U-X4.0 引入）
- 支持段落、粗体、斜体、行内代码、代码块、有序/无序列表
- 代码块使用等宽字体 + `bgSecondary` 背景 + 6px 内边距 + 顶部语言标签
- 链接可点击，外部链接用 `url_launcher` 打开

**Image 块**：
- 使用 `CachedNetworkImage`（已在 U-X4.0 引入）渲染缩略图
- 宽度最大 280px，高宽比自适应
- 圆角 6px
- 点击放大全屏查看（`InteractiveViewer` + 黑色背景）
- 加载中显示灰色骨架屏，加载失败显示裂图图标 + "图片加载失败"

**File 块**：
- 文件卡片：文件图标（按类型：PDF/Word/Excel/通用）+ 文件名 + 文件大小
- 背景 `bgSecondary`，圆角 8px，内边距 12px
- 点击下载或预览（url_launcher 打开链接）

**Message 标记（来自 VE 的 markers）**：
- 气泡底部可选折叠面板：显示意图标签（如"查询天气"）、工作上下文链接
- 使用 `caption1` 字号 + `textTertiary` 颜色

**系统消息**（降级通知/日程提醒）：
- 居中，无气泡背景，`caption1` 字号，`textTertiary` 颜色
- 前面有小图标（信息/警告/日程）

### 3. 消息操作 — `message_actions.dart`

**点击消息气泡**：显示底部操作面板（`showModalBottomSheet`），包含：

| 操作 | 图标 | 说明 |
|------|------|------|
| 复制文字 | `content_copy` | 复制消息文本到剪贴板，SnackBar 提示 |
| 回复 | `reply` | 进入线程回复（或引用回复） |
| 添加反应 | `add_reaction` | 打开 emoji 选择器 |
| 转发 | `forward` | 转发到其他频道/联系人（基础版：生成分享文本） |
| 删除 | `delete` | 仅自己的消息可见，确认弹窗 |

**长按消息气泡**：触发触觉反馈（`HapticFeedback.lightImpact`），然后显示操作面板。

**滑动操作**（参考微信/Telegram）：
- 自己的消息左滑：删除按钮（红色），右滑无效
- 他人的消息左滑无效，右滑：回复按钮

### 4. 输入框区域 — `chat_input_bar.dart`

固定在页面底部，白色背景，顶部 0.5px 分割线：

**布局**（从左到右）：
1. 附件按钮（`+` 圆形图标按钮，36px，`textTertiary` 颜色）
2. 文本输入框：`TextField`，`maxLines: 5`，`minLines: 1`，自动增高
3. 表情按钮（可选，emoji 图标）
4. 发送按钮：`IconButton`，主色，输入框为空时灰色禁用

**附件菜单**（点击 `+` 弹出底部面板）：
- 拍照（摄像头图标）
- 相册（图片图标）
- 文件（文件夹图标）

**输入框行为**：
- 键盘弹出时输入框上移，消息列表自动滚到底部
- 输入框有文字时发送按钮亮起主色，为空时灰色
- 发送后输入框清空，消息列表滚到底部

### 5. 线程面板 — `thread_panel.dart`

从消息气泡的"回复"操作或线程按钮打开：

- 右侧滑入面板（移动端全屏、桌面端宽度 400px）
- 顶部：根消息卡片（完整渲染）+ "线程 · N 条回复"
- 中部：回复列表（按时间正序）
- 底部：回复输入框
- 关闭按钮或左滑返回

### 6. 附件预览 — `attachment_preview.dart`

图片全屏预览：
- 黑色背景，顶部关闭按钮（白色）
- `InteractiveViewer` 支持缩放/平移
- 底部：下载按钮、分享按钮
- 支持左右滑动切换多张图片

### 7. 聊天列表页 — `chat_list_screen.dart`

会话列表，每项含：
- 头像（含在线状态）+ 频道名称 + 最后一条消息摘要（1 行，`textSecondary`）+ 时间（`caption2`）
- 未读消息角标（蓝色数字或红点）
- 右滑：标记已读/置顶/退出频道
- 下拉刷新（RefreshIndicator）
- 搜索栏（点击顶栏搜索图标展开）

## 约束 (Constraints)

- 所有颜色/字号/间距使用 U-9.1 设计 Token
- 图片使用 `CachedNetworkImage`
- 富文本使用 `flutter_markdown`
- 消息列表使用 `ListView.builder` 确保大列表性能
- 键盘处理：使用 `MediaQuery.of(context).viewInsets.bottom` 而非硬编码
