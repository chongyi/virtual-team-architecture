# Phase 9：企业级 UI 全面重塑

> **开始前请先读取**：`CONTEXT.md`、`PREFLIGHT.md`（全局项目上下文与约束）和 `.env-context`（路径变量）。

## 阶段概览

Phase 9 将当前功能可用但视觉粗糙的 Flutter 客户端重塑为**现代化企业协作应用**，对标 WeChat Work（企业微信）、Feishu（飞书）、Slack 的设计品质。9 个单元覆盖设计系统、导航、聊天、联系人、组织、协作工具、动效、深色模式、一致性审计。

## 单元清单

| 序号 | 单元 | 内容 |
|------|------|------|
| 1 | U-9.1 | 设计系统基础：色彩、字体、间距、圆角、阴影、图标 |
| 2 | U-9.2 | 应用壳与导航：响应式布局、底部导航、侧边栏 |
| 3 | U-9.3 | 聊天体验：气泡、输入框、表情、线程、附件、滑动操作 |
| 4 | U-9.4 | 联系人/频道：列表、搜索、在线状态、未读角标 |
| 5 | U-9.5 | 组织管理与个人设置 |
| 6 | U-9.6 | 协作工具 UI：文档/表格/看板/审批/日程 |
| 7 | U-9.7 | 空状态/加载/错误 + 页面动效与微交互 |
| 8 | U-9.8 | 深色模式 + 无障碍访问 |
| 9 | U-9.9 | 全应用视觉一致性审计与修复 |
